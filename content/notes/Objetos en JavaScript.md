Un objeto consiste un conjunto de pares de llave-valor.

La llave es el nombre o etiqueta que se puede usar para referirse al valor.

La siguiente sintaxis es la forma **literal** de escribir de un objeto:

```javaScript
const object = {
	firstName: 'Jaime',
	lastName: 'Ayala'
};
```
